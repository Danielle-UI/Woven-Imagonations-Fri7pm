<p>This project is a simple web-based Crochet Store application that allows users to:

Register and log in

Browse products 

Add items to a personal shopping cart

Proceed to checkout

View Dashboard of user statistics

Generate and view invoices

Search through all invoices in the console

View user-specific invoices

All application data (users, products, cart, invoices) is stored using LocalStorage in the browser</p>
